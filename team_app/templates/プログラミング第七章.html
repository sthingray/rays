第七章

if文（もし～ならば）

コンピュータの強みの一つは、条件判断により「賢い」仕組みを作れることです。
・「雨が降ってきたら、自動的に窓を閉める」
・「メッセージが届いたら、着信音を鳴らす」
・「水温が～℃以上になったら、火を止める」

今回は条件に基づく制御を勉強します。

ある条件が成り立つときに何かを実行する場合は、「if」を用います。
「条件Ａが成り立つならば処理Ｂを実行する」と書くには、以下のような形にします。
if 条件Ａ:
　　処理Ｂ　
^^^^
def a():のようにインデントを忘れずに！

例題１：正しいパスワード「open」を入力した場合だけ秘密のメッセージを表示しよう
回答
＞＞＞password = ‘open’　　　　　　　　　　←パスワードを設定
＞＞＞message = ‘good!’　　　　　　　　　　←秘密のメッセージ設定
＞＞＞if input(‘合言葉は？’) == password:　←input()で入力を受け取り、条件分岐
＞＞＞　　print(message)　　　　　　　　　←print()でmessageの中身を表示

※input(‘表示する内容’)で入力をインプットすることができる。
※print()で値を出力することができる。


例題の中にある“==”は「比較演算子」と呼ばれるものです。
他にも様々な種類があります。
・a == b : aはｂと等しい（数学の＝に相当）
・a != b : aはbと異なる （数学の≠に相当）
・a ＜ b : aはbより小さい
・a ＞ b : aはbより大きい
・a ＞= b : aはb以上である（数学の≧に相当）
・a ＜= b : aはb以下である（数学の≦に相当）

elseとelif

条件が成り立たない場合の分岐を作る場合「else」を使います。
また、if以外の条件で成り立つ分岐を作るには「elif」を使います。

例題２：ph値に対する液性を表すプログラム
回答
＞＞＞ph = float(input(“ph値は？”))
＞＞＞if ph ＜ 7.0:　　　　　　　　←もし(if)なら...
＞＞＞　　print(“酸性です”)
＞＞＞elif ph == 7.0:　　　　　　　←それ以外(else)で、もし(if)なら...
＞＞＞　　print(“中性です”)
＞＞＞else:　　　　　　　　　　　　←それ以外(else)なら...
＞＞＞　　print(“アルカリ性です”)
